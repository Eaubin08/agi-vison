# Contribuer à Obsidia AGI

## 1. Philosophie de Contribution
Obsidia AGI est un projet fondé sur la **Vérifiabilité Universelle**. Nous accueillons les contributions qui renforcent les invariants mathématiques du système.

## 2. Domaines de Contribution
- **Preuves Formelles (Lean 4)** : Renforcement des invariants du Noyau Déterministe.
- **Modélisation (TLA+)** : Validation des protocoles de l'Espace Latent.
- **Agents Périphériques** : Amélioration de la friction cognitive (Douteur, Nettoyeur).
- **Standard X-108** : Évolution des spécifications de gouvernance ex-ante.

## 3. Processus de Validation
Toute contribution doit être accompagnée d'une preuve de non-dérive cognitive. Le Juge Déterministe audite automatiquement chaque Pull Request avant scellement.
